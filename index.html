Discustom

Timeline

- Discustom started as a mini project that was just some CLI status changer, then turned into a UI based application.

- Distatus came along which was the UI based application that was released for easier use.

- A friend as me about RPC and using it, so I found packages and other projects that allow that functionality, so I made Dispresence.

- So I merged them, and Discustom was created.
